Fundamentals of Java

A comprehensive collection of Java programs covering fundamental programming concepts, arrays, loops, basic problem-solving techniques, and pattern printing exercises. This repository serves as a personal learning resource and reference guide for building a strong foundation in Java programming.

Arrays

• Counting Even and Odd Numbers
• Removing Duplicates
• Reversing Arrays

Loops
• Fibonacci Sequence
• Prime Number Programs

Pattern Printing
• Butterfly Pattern

• Number Patterns
